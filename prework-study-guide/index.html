HTML

Hypertext Markup Language (HTML) is used to create the structure of a webpage. It is made up of elements which work together to provide content and the basic framework for all pages on the web. Please read further for more information on the different HTML elements.

HEAD

The HEAD element is a container which contains information, or metadata, that helps the browser to render the page correctly. Elements included within the HEAD are not visible when a user looks at a webpage. The tags used inside a HEAD include:

  • title: element sets the title of the webpage. This information is useful as it is used by Search engines to rank and list your website in search results.
  • base: element defines the base URL for a webpage.
  • link: element connects an external resource to the HTML document.
  • style: element defines the CSS styles associated with a website.
  • meta: element defines metadata about the website. These include:
    • keywords: kw used by search engines.
    • description: description about the website.
    • viewport: Is the user’s visible area of a webpage on a device. It varies from one device to another. It’s smaller on a mobile device and more on a desktop.

BODY

The BODY element contains the content that is shown to the user. All the elements that we want the user to see and interact with must be placed inside this element. Listed below are few of the tags used within the BODY: include:

  • h1...h6: heading elements to represent Title/subtitles. They should appear in order. And only one h1 for the entire page. It is a block level element.
  • p tag: text paragraph. It is a block level element.
  • ul and ol tags: unordered/ordered list
  • header: contains a navigation menu or introductory information about the webpage.
  • main: holds the main content of the webpage.
  • section: elements are semantic elements that define a section in a document that contains thematic content.
  • footer: holds navigation menu, site map, author, copyright.
